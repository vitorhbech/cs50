# CS50 - Harvard University

This repository contains my solutions and exercises from CS50 – Introduction to Computer Science, a course offered by Harvard University.

The course covers fundamental concepts in computer science, including algorithms, memory management, data structures, and software development. The exercises are primarily implemented in C and Python and focus on building problem-solving skills and understanding how software works at a lower level.

# Topics covered

- C programming
- Memory management
- Algorithms
- Data structures
- Python
- SQL
- Web development
- Cryptography basics
